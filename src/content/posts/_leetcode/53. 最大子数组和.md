##解题思路
1.先定义一个cur，以当前位置结尾的最大子数组和
2.定义一个结果ans，接收现在最大的子数组和
3.for循环，遍历完所有的num，但是逻辑是从第一个num开始，后面存储的都是以当前位置结尾的最大子数组和，例如nums[0] = -1, nums[1] = -2,那么cur会是-2而不是-3



二.暴力解法
for (int i = 0; i < n; i++) {
    int sum = 0;
    for (int j = i; j < n; j++) {
        sum += nums[j];
        ans = max(ans, sum);
    }
}

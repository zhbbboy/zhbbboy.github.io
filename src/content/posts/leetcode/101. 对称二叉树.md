### ① 核心痛点与破局点

- 🛑 **痛点**：判断轴对称不能只孤立地看单个节点，而是要比较左右两棵子树是否互为**镜像**。
    
- 🔄 **破局点（双指针联动）**：将根节点的左子树和右子树看作两棵独立的树。遍历时，两根指针必须**成对反向移动**：
    
    - 树 1 的**左**孩子必须与树 2 的**右**孩子相等。
        
    - 树 1 的**右**孩子必须与树 2 的**左**孩子相等。
## 解题思路
### 一.  递归法
1. 首先得创建一个check方法，用来检查左右子树是否对称的。
2. check方法-终止条件：如果左右子树都为空就返回true，如果左右子树有一个为空则返回false，如果左右子树不为空且不相等那就返回false。
3. check方法-递归步骤：如果左右子树不为空且相等的话就继续遍历它们的左右孙子树然后互相比较。
4. 最后核心的接口，首先判断是否为空节点，如果是则返回true，否则就返回check左右子树的结果。


```
class Solution {
public:
    boolean isSymmetric(TreeNode* root) {
        if (root == nullptr) return true;
        return check(root->left, root->right);
    }

private:
    bool check(TreeNode* left, TreeNode* right) {
        // 要素 1：基线条件（出口）
        if (left == nullptr && right == nullptr) return true;
        if (left == nullptr || right == nullptr) return false;
        if (left->val != right->val) return false;
        
        // 要素 2：递归推进（外侧与外侧比，内侧与内侧比）
        return check(left->left, right->right) && check(left->right, right->left);
    }
};
```

### 二.迭代法
使用队列模拟递归的压栈过程，保证每次成对取出需要比较的镜像节点。在 C++ 中，我们直接使用 `std::queue` 即可。
1. 首先判断该树是否为空树，如果是那他就是对称的。
2. 首先该迭代用队列，首先创建一个树节点的队列q。然后将root左子树和右子树指针依次压入队列中。
3. 进入循环，循环条件就是只要队列不为空，先将队列的头节点赋值给u，再pop，然后再将头节点赋值给v，然后再pop。然后对u，v进行判断看是否不相等。若不相等return false，若相等继续遍历比较。

```
#include <queue>

class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        if (root == nullptr) return true;
        
        std::queue<TreeNode*> q;
        q.push(root->left);
        q.push(root->right);

        while (!q.empty()) {
            TreeNode* u = q.front(); q.pop();
            TreeNode* v = q.front(); q.pop();

            // 如果两个节点都为空，说明这一对镜像分支走到头了，且完全对称，继续检查下一对
            if (u == nullptr && v == nullptr) continue;
            
            // 如果只有一个为空，或者值不相等，说明不对称
            if (u == nullptr || v == nullptr) return false;
            if (u->val != v->val) return false;

            // 按镜像顺序成对入队
            q.push(u->left);
            q.push(v->right); // 外侧一对

            q.push(u->right);
            q.push(v->left);  // 内侧一对
        }
        return true;
    }
};
```

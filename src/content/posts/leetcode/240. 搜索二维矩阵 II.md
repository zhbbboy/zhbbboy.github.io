## 解题思路
1. 这题的思路主要是要找到一个特殊点，那就是右上角，因为大于右上角的数在它下面，小于右上角的数在它左边
2. 所以就先定义row和col，它们初始值是与右上角的ij相等。
3. 然后再循环，其循环条件就是如果row< m 的并且col >=0 的那么就可以循环，否则退出循环。
4. 然后如果target>右上角值那就向下移动去寻找，如果小于右上角值就向左移动
5. 如果相等就return true，否则return false



### ③ 面试追问 + 答案

**Q：为什么从左下角出发也可以？**

左下角同样特殊：是最后一行最小值，也是第一列最大值。向右走变大，向上走变小，逻辑对称，同样 O(m+n)。

cpp

```cpp
int row = m - 1, col = 0;
while (row >= 0 && col < n) {
    if (matrix[row][col] == target)  return true;
    else if (matrix[row][col] > target) row--;  // 上移
    else col++;                                  // 右移
}
```

**Q：为什么不能从左上角或右下角出发？**

左上角是最小值：比 target 小只知道右移或下移，两个方向都行，**无法排除任何一行或一列**，退化成暴力搜索。右下角同理（是最大值，比 target 大时左移或上移都可以，方向不唯一）。

**Q：和 74 题"搜索二维矩阵 I"有什么区别？**

74 题的矩阵是严格有序的（每行末尾小于下行开头），整个矩阵可以展开成一个有序数组做二分，O(log(mn))。这道题行列独立有序，不能展开，只能用 O(m+n) 的右上角法。

## 解题思路（动态规划 / 贪心）

1. 先定义一个 `cur`，表示以当前位置结尾的最大子数组和。
2. 定义一个结果 `ans`，接收目前为止最大的子数组和。
3. for 循环遍历完所有的 num，逻辑是从第一个 num 开始，`cur` 存的始终是以当前位置结尾的最大子数组和。例如 `nums[0] = -1, nums[1] = -2`，那么 `cur` 会是 -2 而不是 -3。

## 暴力解法

```cpp
for (int i = 0; i < n; i++) {
    int sum = 0;
    for (int j = i; j < n; j++) {
        sum += nums[j];
        ans = max(ans, sum);
    }
}
```

## 解题思路

1. `need[c]`：t 中字符 c 需要出现几次；`window[c]`：当前窗口中字符 c 出现了几次。
2. 首先先遍历出 t 的所有字符并记录到 need 当中去。
3. left 是窗口的左边界，count 是指 window 中满足 t 要求的字符数量，minlen 记录当前最短窗口子串长度，start 是结果窗口的起始位置。
4. 在 s 中移动窗口右指针 right，每遍历到一个字符就先存储到 c 中。如果 t 中含有 c，那就 `window[c]++`；如果 window 中 c 的数量不超过 need 中 c 的数量，那就 `count++`。意思就是：如果该字符是 t 需要的、并且不是多出来的，就记录到 count 中。如此不断迭代。
5. 当 window 刚好迭代到包含 t 后，先把长度记录一下，跟 minlen 比较，若小于 minlen，那它就是现在的最短窗口子串，令 `start = left`。
6. 然后再开始移动左指针 left。若 window 还是包含 t，就继续第 5 步；若不是，就继续第 4 步。
7. 如果 minlen 没变，也就是 s 里面没有包含 t 的子串，则返回 `""`。

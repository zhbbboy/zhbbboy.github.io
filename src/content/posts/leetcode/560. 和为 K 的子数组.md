## 解题思路：前缀和 + 哈希表

这题要求统计和为 `k` 的连续子数组个数。由于数组中可能有负数，不能简单使用滑动窗口，因此使用前缀和。

设当前遍历到某个位置时，当前前缀和为 `sum`。如果之前存在某个前缀和 `oldSum`，使得：

```text
sum - oldSum = k
```

那么从 `oldSum` 对应位置的后一个元素，到当前位置这一段连续子数组的和就是 `k`。

将公式变形：

```text
oldSum = sum - k
```

所以每次遍历到当前位置时，只需要查看之前是否出现过前缀和 `sum - k`。

用哈希表 `mp` 记录：

```text
mp[前缀和] = 这个前缀和在当前位置之前出现过的次数
```

遍历过程中：

1. 先更新当前前缀和 `sum += num`。
2. 查找 `mp[sum - k]`，如果存在，说明有若干个以当前位置结尾、和为 `k` 的子数组，将其次数加入答案。
3. 再执行 `mp[sum]++`，把当前前缀和记录下来，供后面的元素使用。

初始化：

```cpp
mp[0] = 1;
```

表示在遍历开始前，前缀和 0 已经出现过 1 次。这样可以处理从下标 0 开始、和正好为 `k` 的子数组。

## 核心代码

```cpp
unordered_map<int, int> mp;
mp[0] = 1;

int sum = 0;
int ans = 0;

for (int num : nums) {
    sum += num;

    if (mp.count(sum - k)) {
        ans += mp[sum - k];
    }

    mp[sum]++;
}

return ans;
```

## 核心记忆

```text
当前前缀和是 sum。
想找和为 k 的子数组，
就看之前有多少个前缀和等于 sum - k。
```

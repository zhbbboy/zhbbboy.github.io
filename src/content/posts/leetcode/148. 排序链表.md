### ① 为什么必须用归并排序？

O(n log n) 的排序算法有快排、归并、堆排。但对链表来说：

快排需要随机访问（链表做不到，效率退化），堆排同理。**归并排序天然适合链表**——它只需要"分割"和"合并"，这两个操作链表都能高效完成。所以链表排序的标准答案就是归并排序。

---

### ② 归并排序的整体思路：分治

归并排序就是经典的"分而治之"：

```
① 分（divide）：把链表从中间断开，分成左右两半
② 治（conquer）：递归地对左右两半分别排序
③ 合（combine）：把两个已排序的半段合并成一个有序链表
```

递归地分到每段只剩一个节点（单个节点天然有序），再逐层向上合并。

## 解题思路
### 一.递归法（自顶向下递归）
1. 首先得多写两个辅助函数，一个是找中点的辅助函数findMind，第二个是合并两个链表的辅助函数merge。
2. 主函数流程-终止条件：如果head或者head->next为空的话就说明只有一个节点或者没有节点说明已经排序好了直接返回头节点
3. 主流程函数-递归前的操作：先找出中间节点然后再将它们拆分
4. 主流程函数-信任递归：将左边与右边的链表调用完的排序函数的结果分别返回赋值给left和right两个指针。最后直接返回合并完的两个指针
### ③ 完整 C++ 代码（自顶向下递归）

```cpp
class Solution {
public:
    ListNode* sortList(ListNode* head) {
        // 终止条件：空链表或单节点，已经有序
        if (head == nullptr || head->next == nullptr) return head;
        
        // ① 分：快慢指针找中点，从中间断开
        ListNode* mid = findMid(head);
        ListNode* right = mid->next;
        mid->next = nullptr;  // 切断，分成两条独立链表
        
        // ② 治：递归排序左右两半
        ListNode* left = sortList(head);
        ListNode* rightSorted = sortList(right);
        
        // ③ 合：合并两个有序链表（复用 21 题）
        return merge(left, rightSorted);
    }
    
private:
    // 找中点（快慢指针），返回前半段的最后一个节点
    ListNode* findMid(ListNode* head) {
        ListNode* slow = head;
        ListNode* fast = head->next;  // 注意 fast 从 head->next 开始
        while (fast != nullptr && fast->next != nullptr) {
            slow = slow->next;
            fast = fast->next->next;
        }
        return slow;  // slow 是前半段的最后一个节点
    }
    
    // 合并两个有序链表
    ListNode* merge(ListNode* l1, ListNode* l2) {
        ListNode dummy(0);
        ListNode* tail = &dummy;
        while (l1 != nullptr && l2 != nullptr) {
            if (l1->val <= l2->val) {
                tail->next = l1;
                l1 = l1->next;
            } else {
                tail->next = l2;
                l2 = l2->next;
            }
            tail = tail->next;
        }
        tail->next = (l1 != nullptr) ? l1 : l2;
        return dummy.next;
    }
};
```

---

### ④ 一个关键细节：findMid 里 fast 为什么从 `head->next` 开始？

这里和找中点（876题）有一个微妙区别。如果 `fast` 从 `head` 开始，对于两个节点 `[4,2]`：

```
fast 从 head 开始：slow 会停在第二个节点（2），mid->next 切断后右半为空
→ 左半 [4,2] 右半 []，没有真正分割！导致无限递归栈溢出
```

让 `fast` 从 `head->next` 开始，slow 会停在**前半段的最后一个**：

```
[4,2]：slow 停在 4，mid=4，right=2，切成 [4] 和 [2]，正确分割
```

**口诀：归并排序找中点，要让中点偏左，保证两半都非空，避免死循环。**

---

### ⑤ 面试常见追问

**Q：这个解法是 O(1) 空间吗？**

严格说不是。自顶向下递归用了 O(log n) 的递归栈空间。要真正做到 O(1) 空间，需要**自底向上**的归并（迭代版），不用递归栈，但代码复杂很多。面试中自顶向下递归通常已经被接受，如果面试官严格要求 O(1)，再提自底向上方案。

**Q：时间复杂度为什么是 O(n log n)？**

分割是 log n 层（每次对半），每一层的合并操作总共遍历 n 个节点，所以是 n × log n。

**Q：为什么不能用快速排序？**

快排依赖随机访问来做 partition（取中间元素、左右交换），链表无法 O(1) 随机访问，导致快排在链表上效率退化且实现复杂。归并只需顺序访问，更适合链表。

### 二.迭代法（自底向上迭代归并）
### 自底向上迭代归并：不用递归栈

**核心思路转变：** 递归是"先一路切到单节点，再往上合并"。迭代反过来——**直接从最小的子段开始，一轮轮往大了合并**。

```
第1轮：每段长度1，两两合并 → 得到一堆长度2的有序段
第2轮：每段长度2，两两合并 → 得到一堆长度4的有序段
第3轮：每段长度4，两两合并 → ...
直到段长 >= 链表总长，排序完成
```

每一轮的段长是 1, 2, 4, 8... 倍增，所以总共 log n 轮，每轮遍历 n 个节点，依然是 O(n log n)。但**没有递归，所以空间是 O(1)**。

### 自底向上完整代码

```cpp
class Solution {
public:
    ListNode* sortList(ListNode* head) {
        if (head == nullptr || head->next == nullptr) return head;
        
        // 先求链表总长度
        int n = 0;
        for (ListNode* p = head; p != nullptr; p = p->next) n++;
        
        ListNode dummy(0);
        dummy.next = head;
        
        // 段长从 1 开始，每轮翻倍
        for (int step = 1; step < n; step *= 2) {
            ListNode* prev = &dummy;   // 已合并部分的尾
            ListNode* cur = dummy.next; // 待处理部分的头
            
            while (cur != nullptr) {
                // 切出第一段（长度 step）
                ListNode* left = cur;
                ListNode* right = split(left, step);
                // 切出第二段（长度 step），cur 移到第三段
                cur = split(right, step);
                
                // 合并 left 和 right，接到 prev 后面
                prev = merge(left, right, prev);
            }
        }
        
        return dummy.next;
    }
    
private:
    // 从 head 往后数 step 个节点切断，返回剩余部分的头
    ListNode* split(ListNode* head, int step) {
        if (head == nullptr) return nullptr;
        for (int i = 1; head->next != nullptr && i < step; i++) {
            head = head->next;
        }
        ListNode* rest = head->next;
        head->next = nullptr;  // 切断
        return rest;
    }
    
    // 合并两个有序链表，接到 prev 后面，返回合并后的尾节点
    ListNode* merge(ListNode* l1, ListNode* l2, ListNode* prev) {
        ListNode* cur = prev;
        while (l1 != nullptr && l2 != nullptr) {
            if (l1->val <= l2->val) {
                cur->next = l1;
                l1 = l1->next;
            } else {
                cur->next = l2;
                l2 = l2->next;
            }
            cur = cur->next;
        }
        cur->next = (l1 != nullptr) ? l1 : l2;
        // 移动到合并后这一段的末尾
        while (cur->next != nullptr) cur = cur->next;
        return cur;
    }
};
```
